# Todo App with Priority

A simple **Todo App** built using **Vanilla JavaScript, HTML & CSS**.  
This app allows you to **add tasks, mark them as complete, delete tasks, and set priority levels**. Tasks are stored in **LocalStorage**, so they persist even after refreshing the page.

---

## **Features**

- Add tasks ✅  
- Mark tasks complete with a ✔ tick  
- Delete individual tasks 🗑️  
- Clear all tasks at once  
- Task counter showing incomplete tasks  
- Priority levels: High 🔴, Medium 🟠, Low 🟢  
- Responsive design (desktop + mobile)  
- Persistent data using LocalStorage
